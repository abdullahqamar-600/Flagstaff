Campaigns · Design system

Campaign detail components

Eleven components used across the campaign detail views — an approval drawer, a generated-post card and its loading state, the campaign detail panel, a compact horizontal post row, an approval drawer variant with post analytics, four popups (date + time picker, dropdown options, campaign disclaimer, rejection feedback), and a read-only banner.

Picking these up in another page?   Import designs/_campaigns-components.css — it defines module tokens (--camp-*) and every class shown here. Written spec (naming, do's & don'ts, when-to-use) lives at design-system/campaigns.md.
